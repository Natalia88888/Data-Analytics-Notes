# Справочник по аналитике в Excel и Google Sheets

> **Статус:** В процессе активного изучения и применения в проектах.  
> **Цель:** Систематизация знаний для быстрой обработки данных и построения бизнес-отчетов.

## 1. Фундамент: Гигиена данных
Прежде чем приступать к анализу, я проверяю данные на соответствие стандартам:
*   **Атомарность:** Каждая ячейка содержит только один смысл (например, Фамилия и Имя разделены).
*   **Чистота типов:** В одном столбце либо только цифры, либо только даты. Это критично для корректной работы формул.
*   **Умные таблицы (Ctrl + T):** Преобразование обычного диапазона в таблицу для автоматического подтягивания формул в новые строки.

## 2. Ключевые функции (Top Tier)
Инструменты, которые я использую для связи данных и расчетов:

| Функция | Зачем нужна (SQL аналог) | Комментарий |
| :--- | :--- | :--- |
| **XLOOKUP** | `JOIN` | Находит данные в другой таблице. Безопаснее и быстрее, чем устаревший ВПР. |
| **SUMIFS** | `SUM` + `WHERE` | Считает сумму, если соблюдается одно или несколько условий. |
| **COUNTIFS** | `COUNT` + `WHERE` | Считает количество строк по заданным критериям. |
| **IF / IFS** | `CASE WHEN` | Позволяет создавать сложные логические ветвления. |

## 3. Сводные таблицы (Pivot Tables)
Это мой основной инструмент для превращения сырых данных в ответы на бизнес-вопросы:
*   **Агрегация:** Мгновенный подсчет сумм, средних значений и количеств.
*   **Группировка:** Превращение списка дат в отчет по месяцам или неделям.
*   **Слайсеры (Slicers):** Создание удобных кнопок-фильтров для визуального управления отчетом.

## 4. Визуализация и Dashboards
*   **Условное форматирование:** Автоматическая подсветка аномалий (например, падение продаж ниже нормы).
*   **Sparklines:** Микро-графики прямо в ячейках для отслеживания трендов без построения больших диаграмм.
*   **Валидация (Data Validation):** Создание выпадающих списков, чтобы исключить ошибки ручного ввода.

## 5. Обработка ошибок и контроль качества
Профессиональный отчет должен быть защищен от системных сбоев:
*   **IFERROR:** Обязательная «обертка» для формул поиска, чтобы заменить ошибки типа `#N/A` на чистые пропуски или нули.
*   **ISBLANK / ISERROR:** Функции проверки ячеек перед проведением массовых вычислений.
*   **Защита листов:** Блокировка ячеек с формулами от случайного изменения (критично при совместной работе в Google Sheets).
