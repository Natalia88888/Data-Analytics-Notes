# 📊 Полный гид по базовой статистике

Этот документ объединяет ключевые концепции статистики: от базовых терминов до проверки гипотез. Идеально подходит для быстрого повторения и использования в качестве справочника.

---

## 1. Базовая статистика: Основы

### 1.1 Основные понятия и термины
Статистика — это наука о сборе, анализе и интерпретации данных.

*   **Генеральная совокупность (Population):** Полный набор всех объектов, которые мы хотим изучить (например, *все* жители страны).
*   **Выборка (Sample):** Часть объектов, выбранных из генеральной совокупности для исследования (например, *1000* опрошенных жителей).
*   **Параметр:** Числовая характеристика генеральной совокупности (обозначается греческими буквами, например, $\mu$ — среднее).
*   **Статистика:** Числовая характеристика выборки (обозначается латинскими буквами, например, $\bar{x}$ — среднее).
*   **Переменные:**
    *   *Количественные:* Числа (рост, доход, температура).
    *   *Качественные (номинальные):* Категории (цвет глаз, марка машины, пол).

### 1.2 Меры центральной тенденции
Показывают, где находится "центр" данных (типичное значение).

| Мера | Определение | Когда использовать |
| :--- | :--- | :--- |
| **Среднее ($\bar{x}$)** | Сумма всех значений, деленная на их количество. | Для симметричных данных без выбросов. |
| **Медиана** | Значение, которое делит упорядоченный список пополам. | Если в данных есть сильные перекосы или выбросы. |
| **Мода** | Значение, которое встречается чаще всего. | Для качественных данных (категорий). |

> **💡 Пример:** В наборе `[1, 2, 2, 3, 100]`:
> *   Среднее = 21.6 (сильно искажено числом 100).
> *   Медиана = 2 (лучше отражает типичное число).

### 1.3 Меры изменчивости (разброса)
Показывают, насколько данные "раскиданы" относительно центра.

1.  **Размах (Range):** Разница между максимумом и минимумом ($Max - Min$).
2.  **Дисперсия ($\sigma^2$):** Средний квадрат отклонения от среднего. Показывает общую вариативность.
3.  **Стандартное отклонение ($\sigma$):** Квадратный корень из дисперсии.
    *   *Смысл:* На сколько в среднем каждое значение отклоняется от центра.
    *   *Единицы:* Измеряется в тех же единицах, что и данные (градусы, рубли и т.д.).
4.  **Квартили:** Делят данные на 4 части (25%, 50%, 75%).
    *   *Межквартильный размах (IQR):* Разница между 75-м и 25-м процентилем. Помогает находить выбросы.

### 1.4 Форма распределения данных

#### 1. Нормальное распределение (Гаусса)
*   **Вид:** Симметричный колокол.
*   **Свойство:** Среднее = Медиана = Мода.
*   **Правило 3-х сигм:**
    *   ~68% данных лежат в пределах $1\sigma$.
    *   ~95% данных лежат в пределах $2\sigma$.
    *   ~99.7% данных лежат в пределах $3\sigma$.

#### 2. Асимметрия (Skewness)
*   **Положительная (правосторонняя):** "Хвост" графика длиннее справа. Большинство данных слева (маленькие значения).
*   **Отрицательная (левосторонняя):** "Хвост" длиннее слева. Большинство данных справа (большие значения).

#### 3. Эксцесс (Kurtosis)
Показывает "остроту" пика и "толщину" хвостов.
*   *Высокий пик:* Данные сильно сгруппированы в центре.
*   *Низкий пик:* Данные сильно разбросаны.

## 2. t-распределение Стьюдента

### 2.1 Что это такое?
Распределение, используемое для оценки параметров генеральной совокупности на основе **маленьких выборок**.

*   **Автор:** Вильям Госсет (псевдоним "Student"), 1908 г.
*   **Главная цель:** Учесть неопределенность и риск ошибки, когда данных мало.

> **Суть:** Если Z-распределение — это «идеальный» колокол для огромных данных, то t-распределение — это «осторожный» колокол для маленьких групп.

### 2.2 Геометрия графика (Толстые хвосты)

| Характеристика | t-распределение | Нормальное (Z) |
| :--- | :--- | :--- |
| **Форма** | Колоколообразная, симметричная | Колоколообразная, симметричная |
| **Высота** | Более низкий и плоский пик | Высокий и острый пик |
| **Хвосты (края)** | **Толстые** (высокая вероятность крайностей) | **Тонкие** (крайности почти невозможны) |

**Почему хвосты толстые?** Это статистическая «страховка». При малых данных мы допускаем, что случайные экстремальные значения могут встречаться чаще.

### 2.3 Степени свободы (Degrees of Freedom, $df$)
Параметр, определяющий форму t-распределения: $df = n - 1$, где $n$ — объем выборки.

*   **Мало $df$:** График низкий, хвосты максимально толстые.
*   **Рост $df$:** Пик растет вверх, хвосты «худеют».
*   **Предел ($df \to \infty$):** При $n > 30$ (или 100) t-распределение становится практически идентичным **нормальному**.

## 3. Доверительные интервалы (CI)

### 3.1 Что это такое?
**Доверительный интервал** — это диапазон, в котором с определенной вероятностью находится истинное среднее значение генеральной совокупности.

> **Пример:** Вместо "Средний рост точно 175 см", мы говорим: "Я на 95% уверен, что средний рост находится в пределах от 172 до 178 см".

### 3.2 Уровень доверия (Confidence Level)
*   **95% (стандарт):** Готовы ошибиться в 5 случаях из 100.
*   **99% (строгий):** Ошибка лишь в 1 случае из 100.

### 3.3 Зависимости размера интервала

| Фактор | Изменение | Влияние на интервал |
| :--- | :--- | :--- |
| **Уровень доверия** | Увеличиваем (95% → 99%) | **Расширяется** (нужно больше места для уверенности) |
| **Размер выборки ($n$)** | Увеличиваем данные | **Сужается** (мы становимся точнее) |
| **Разброс ($\sigma$)** | Данные сильно "скачут" | **Расширяется** (труднее угадать точно) |

## 4. Проверка гипотез (Hypothesis Testing)

### 4.1 Основные понятия
1.  **Нулевая гипотеза ($H_0$):** "Эффекта нет", "различия случайны". (Лекарство не работает).
2.  **Альтернативная гипотеза ($H_1$):** "Эффект есть!", "различия реальны". (Лекарство лечит).

### 4.2 Ошибки I и II рода

| Ошибка | Суть | Метафора (Суд) |
| :--- | :--- | :--- |
| **I род ($\alpha$)** | Найти эффект там, где его нет | Осудить невиновного |
| **II род ($\beta$)** | Пропустить реальный эффект | Отпустить преступника |

### 4.3 P-значение (P-value)
Вероятность получить такой результат случайно, если верна нулевая гипотеза ($H_0$).

*   **$P < 0.05$:** Результат **статистически значим**. Отвергаем $H_0$. Эффект скорее всего реален.
*   **$P > 0.05$:** Не можем отвергнуть $H_0$. Различия, скорее всего, случайны.

> **Золотое правило:** Чем меньше P-value, тем меньше шансов, что результат — это просто везение.
